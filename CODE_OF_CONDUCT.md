# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/) version 2.1.

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Expected behavior

- Be respectful
- Be constructive
- Focus feedback on the work

## Unacceptable behavior

- Harassment
- Hate speech
- Personal attacks
- Doxxing or threats

## Enforcement

Use GitHub Discussions or a private security advisory if you need maintainer intervention.
