# Contributing

Thanks for helping improve OpenCode Overlay.

## Start here

- Read `AGENTS.md` first.
- Check existing Discussions and Issues before opening new ones.
- Keep changes in `src/` whenever possible.

## For code or behavior changes

- Run `bun run check`
- Run `bun run test`
- Run `bun run build` if you touched build/runtime code

## Review checklist

- Source files were changed, not generated output
- Docs were updated when workflow expectations changed
- The change is easy to review and limited in scope
