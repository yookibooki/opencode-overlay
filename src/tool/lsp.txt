LSP code intelligence. Supported ops:

- `goToDefinition`
- `findReferences`
- `hover`
- `documentSymbol`
- `workspaceSymbol`
- `goToImplementation`
- `prepareCallHierarchy`
- `incomingCalls`
- `outgoingCalls`

Requires:
- `filePath`
- `line` (1-based)
- `character` (1-based)
  Errors if no server is configured.
